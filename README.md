# videojs-unmute
Video.js plugin to add automatically a Unmute button like Youtube when a video is set to autoplay

## Basic Usage  

Call the plugin: 

    const myPlayer = videojs("vid1");
    volumeLevel = 1;
    // Call the plugin
    myPlayer.unmuteButton();
